## Reporting a Vulnerability

We take the security of Reefline seriously. If you believe you have found a security vulnerability in Reefline, please report it to us as described below.

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please report them via email to:

`siddhantprateek@agentkube.com`

You should receive a response within 48 hours. If for some reason you do not, please follow up via email to ensure we received your original message.

### What to Include

* A clear description of the potential vulnerability.
* Steps to reproduce the issue.
* Any relevant code snippets or configuration files.

Thank you for helping keep Reefline safe!
